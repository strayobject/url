About

League\Url is a simple library to ease URL parsing and editing in PHP. The goal of the library is to be powerful while remaining simple to use.

URL on Packagist

Highlights

  • Simple API
  • Urls can be treated as Value Objects or as Immutable Value Objects
  • Composer ready and PSR-2 and PSR-4 compliant
  • PHP 5.3 as a minimal requirement
